# EventHub

Plataforma profissional de agenda pública de eventos.

## Estrutura do Projeto

- `frontend/`: Interface do usuário (HTML, CSS, JS)
- `backend/`: API e lógica de negócio (Node.js, Express)
- `index.html`: Página inicial (referência visual)

## Tecnologias
- Node.js + Express
- PostgreSQL
- Autenticação segura (bcrypt, JWT)
- Layout moderno, responsivo, animações de partículas

## Instruções
O projeto será desenvolvido em etapas, priorizando design, funcionalidades e integração.
